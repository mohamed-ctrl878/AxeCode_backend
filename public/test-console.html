HttpOnly Cookies - Console Testing

Test Commands

Open your browser's Developer Tools (F12) and run these commands in the Console:

1. Login

await testLogin('your-email@example.com', 'your-password')

2. Check Authentication

await testAuth()

3. Get Protected Data

await getProtectedData()

4. Check Current User

await getCurrentUser()

5. Logout

await logout()

6. Check Cookies (Security Test)

checkCookies()
